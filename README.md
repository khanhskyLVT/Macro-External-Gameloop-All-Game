## 💡 Tổng quan | Overview

### [EN] Overview
**Macro External GameLoop** is a recoil control utility specifically optimized for games running on the **GameLoop Emulator** (PUBG Mobile, Free Fire, COD Mobile, etc.). Operating as an **External** mechanism (outside the emulator process), it delivers optimized performance while avoiding direct memory manipulation.

#### Key Features:
* **GameLoop Optimized:** Designed specifically to work seamlessly with the GameLoop Android emulator.
* **Recoil Reduction:** Maintains stable crosshair control during continuous firing.
* **Multi-Game Support:** Easily customizable and applicable across various games on the emulator.
* **Lightweight & Smooth:** Runs efficiently in the background without causing emulator lag or FPS drops.
* **Flexible Configuration:** Effortlessly adjust parameters to match your mouse sensitivity and weapon setup.

* **Giảm giật tối đa:** Kiểm soát góc nảy súng khi xả đạn cực kỳ ổn định.
* **Hỗ trợ đa tựa game:** Dễ dàng tùy chỉnh cho nhiều game khác nhau trên giả lập.
* **Nhẹ & Mượt:** Hoạt động ngầm tối ưu, không làm giật lag hay giảm FPS của giả lập.
